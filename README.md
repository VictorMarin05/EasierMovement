# Control de Sistema Robótico mediante MPU6050

Este repositorio contiene el código fuente desarrollado para mi Trabajo de Investigación (TDR) de 2º de Bachillerato durante el curso 2021-2022 (Prototipo de una silla de ruedas que se mueve mediante los movimientos de la cabeza). El proyecto se centra en la implementación de un sistema de control mediante los movimientos de la cabeza para una silla de ruedas utilizando un acelerómetro y giroscopio.

## Reconocimientos y Premios

El proyecto fue desarrollado en el **Institut Torre Vicens** y recibió los siguientes reconocimientos:
* **Calificación final:** 10/10.
* **Premio extraordinario:** Otorgado por el Institut Torre Vicens al mejor trabajo de investigación de la promoción.
* **3r Premio de la Universitat de Lleida (UdL):** Galardonado en el certamen de trabajos de investigación de Bachillerato a nivel universitario.

## Descripción Técnica

El software está diseñado para interpretar los datos de un acelerómetro y giroscopio **MPU6050** en tiempo real, transformando la inclinación en los ejes X e Y en comandos de movimiento.
